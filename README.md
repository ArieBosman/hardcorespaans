HARDCORESPAANS – Spaanse werkwoordentrainer (Hardcore Mode)
HARDCORESPAANS is het zwaarste niveau binnen de Spaanse werkwoordentrainers. Deze mode is bedoeld voor leerlingen die alle eerdere niveaus beheersen en klaar zijn voor de ultieme uitdaging. In deze trainer oefen je werkwoorden die bekendstaan om hun complexe en onvoorspelbare vormen: volledig onregelmatige stammen, dubbele onregelmatigheden, afwijkende perfecto‑vormen, uitzonderlijke gerundio’s en werkwoorden die in meerdere tijden compleet van patroon veranderen. Denk aan werkwoorden zoals ir, ser, venir, decir, dormir, morir, poner, hacer en meer.

De app kiest automatisch een willekeurig werkwoord, een persoon en een tijd. Jij vult de juiste vervoeging in en krijgt direct feedback. Daarna maak je een voorbeeldzin om de vorm ook in context te trainen. HARDCORESPAANS is ideaal voor A2–B1‑leerlingen die hun grammaticale reflexen willen aanscherpen en willen leren omgaan met de meest lastige uitzonderingen van het Spaans.

Bestand:  
hardcorespaans_app.py

Lokaal starten:

Installeer Streamlit:
pip install streamlit

Start de trainer:
streamlit run hardcorespaans_app.py

HARDCORESPAANS is ontworpen voor maximale uitdaging en groei. Als je dit niveau beheerst, ben je klaar voor vrijwel elke werkwoordsvorm die je in het Spaans tegenkomt.
